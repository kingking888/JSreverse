## 大部分解密基本使用

### 1、环境

node(安装相关库：express等)

python3

###  2、文件说明
jm.js是js的解密文件

server.js是node express运行的接口服务器，提供解密参数接口调用获取

demo.py是python运行的测试文件，请求解密接口以及数据接口，返回数据

###  3、运行

node server.js(服务器上可以使用pm2管理运行)

python demo.py
